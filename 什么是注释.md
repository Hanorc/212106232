# 什么是注释
+ 注释是写在程序中对代码进行解释说明的文字，方便自己和其他人查看，以便理解程序的。
## ==注释有哪些==
+ 单行注释
// 注释内容，只能写一行
+ 多行注释
/* 
	注释内容1
	注释内容2
*/
+ 文档注释：
文档注释的内容是可以提取到一个程序说明文档中去
/**
	注释内容
	注释内容
*/
## 注释的特点
==1.注释不影响程序的执行==
